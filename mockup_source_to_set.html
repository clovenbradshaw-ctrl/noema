Two Modes, One Goal:
Wizard — Click-through UI for simple filter + column select. No code needed.
Query — Full SQL with CREATE SET syntax, aggregations, joins. Plus natural language option ("Show me all California images with low cloud cover") that generates SQL.
Both preserve provenance chain: the new Set knows exactly which Source it came from.
